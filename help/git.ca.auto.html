S'ha activat el login amb Git?
Si es defineix en Sí , l’usuari tindrà accés a lectura / escriptura a tots els dipòsits de Git seleccionats al camp següent. Git és un sistema de control de codi font, normalment utilitzat pels desenvolupadors de programari.

Els usuaris amb accés a qualsevol dipòsit també podran utilitzar GitWeb per navegar per l’origen i els canvis a tots els dipòsits d’aquest domini.
