Git přihlášení povoleno?
Pokud je tato možnost nastavena na Ano , bude uživateli uděleno oprávnění ke čtení a zápisu ke všem repozitářům Git vybraným v níže uvedeném poli. Git je systém řízení zdrojového kódu, který obvykle používají vývojáři softwaru.

Uživatelé s přístupem do jakýchkoli repozitářů budou také moci používat GitWeb k prohlížení zdroje a změn všech repozitářů v této doméně.
