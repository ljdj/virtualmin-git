เปิดใช้งานการเข้าสู่ระบบ Git หรือไม่
เมื่อตั้งค่าเป็น ใช่ ผู้ใช้จะได้รับสิทธิ์เข้าถึงแบบอ่าน / เขียนไปยังที่เก็บ Git ทั้งหมดที่เลือกในฟิลด์ด้านล่าง Git เป็นระบบควบคุมซอร์สโค้ดซึ่งโดยทั่วไปจะใช้โดยนักพัฒนาซอฟต์แวร์

ผู้ใช้ที่มีสิทธิ์เข้าถึงที่เก็บใด ๆ จะสามารถใช้ GitWeb เพื่อเบราว์เซอร์แหล่งที่มาและการเปลี่ยนแปลงที่เก็บ ทั้งหมด ในโดเมนนี้
