Git saioa aktibatuta dago?
Hori Bai aukeratzen, erabiltzaileak emango dira irakurri / idatzi Git biltegiak guztiak azpiko eremuan hautatutako sarbidea. Git iturburu kodea kontrolatzeko sistema da, software garatzaileek normalean erabiltzen dutena.

Edozein biltegietarako sarbidea duten erabiltzaileek GitWeb erabili ahal izango dute iturria arakatzeko eta domeinu honetako biltegi guztietara aldatzeko.
