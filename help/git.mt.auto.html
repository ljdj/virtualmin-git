Login tal-Git attivat?
Meta dan ikun issettjat għal Iva , l-utent jingħata aċċess għall-qari / kitba għar-repożitorji kollha tal-Git magħżula fil-qasam hawn taħt. Git hija sistema ta 'kontroll tal-kodiċi tas-sors, użata tipikament mill-iżviluppaturi tas-softwer.

Utenti b'aċċess għal kwalunkwe repożitorju jkunu jistgħu wkoll jużaw GitWeb biex jibbrawżjaw is-sors u l-bidliet fir - repożitorji kollha f'dan id-dominju.
