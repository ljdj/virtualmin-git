Gewähren Sie Zugriff auf Git-Repositorys
Wenn Git im obigen Feld aktiviert ist, können Sie in diesen Listen auswählen, auf welche spezifischen Repositorys der Benutzer Lese- / Schreibzugriff haben soll. Um ein Repository hinzuzufügen, wählen Sie es aus dem linken Menü aus und klicken Sie auf die Schaltfläche mit dem Rechtspfeil.
