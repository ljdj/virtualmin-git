Želite li dopustiti Git repozitoreje?
Kada je ova opcija omogućena, vlasnik ovog virtualnog poslužitelja moći će stvoriti jedno ili više Git spremišta povezanih s domenom za spremanje izvornog koda nadziranog inačicom i drugih datoteka. Spremišta dostupna svakom korisniku mogu se konfigurirati na stranici korisnika e-pošte / FTP-a.
