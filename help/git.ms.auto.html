Log masuk Git diaktifkan?
Apabila ini diset ke Ya , pengguna akan diberi akses membaca / menulis ke semua repositori Git yang dipilih dalam bidang di bawah ini. Git adalah sistem kawalan kod sumber, biasanya digunakan oleh pembangun perisian.

Pengguna dengan akses ke repositori mana pun juga akan dapat menggunakan GitWeb untuk menyemak imbas sumber dan perubahan pada semua repositori dalam domain ini.
