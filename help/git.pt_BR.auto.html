Login Git ativado?
Quando definido como Sim , o usuário terá acesso de leitura / gravação a todos os repositórios Git selecionados no campo abaixo. Git é um sistema de controle de código fonte, normalmente usado por desenvolvedores de software.

Usuários com acesso a qualquer repositório também poderão usar o GitWeb para pesquisar a fonte e alterar todos os repositórios desse domínio.
