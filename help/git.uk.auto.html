Увімкнено вхід у Git?
Якщо для цього встановлено значення Так , користувачеві буде надано доступ для читання / запису до всіх сховищ Git, вибраних у полі нижче. Git - це система управління вихідним кодом, яка зазвичай використовується розробниками програмного забезпечення.

Користувачі, які мають доступ до будь-яких сховищ, також зможуть використовувати GitWeb для перегляду джерела та змін у всіх сховищах цього домену.
