Accorder l'accès aux référentiels Git
Lorsque Git est activé dans le champ ci-dessus, ces listes vous permettent de sélectionner les référentiels spécifiques auxquels l'utilisateur aura accès en lecture / écriture. Pour ajouter un référentiel, sélectionnez-le dans le menu de gauche et cliquez sur le bouton flèche droite.
