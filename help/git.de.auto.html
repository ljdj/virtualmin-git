Wenn dies auf Ja gesetzt ist , erhält der Benutzer Lese- / Schreibzugriff auf alle im Feld unten ausgewählten Git-Repositorys. Git ist ein Quellcodeverwaltungssystem, das normalerweise von Softwareentwicklern verwendet wird.

Benutzer mit Zugriff auf Repositorys können mit GitWeb auch die Quelle durchsuchen und Änderungen an allen Repositorys in dieser Domäne vornehmen.
