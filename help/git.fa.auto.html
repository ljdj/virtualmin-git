ورود به سیستم Git فعال شده است؟
با تنظیم این گزینه بله ، به همه مخازن Git انتخاب شده در قسمت زیر دسترسی کاربر را می خواند و می نویسد. Git یک سیستم کنترل کد منبع است که معمولاً توسط توسعه دهندگان نرم افزار استفاده می شود.

همچنین کاربرانی که به هر مخازن دسترسی دارند ، می توانند از GitWeb برای مرور منبع و تغییر در تمام مخازن این دامنه استفاده کنند.
