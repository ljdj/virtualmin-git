Vai ir iespējota pieteikšanās?
Ja tas ir iestatīts uz Jā , lietotājam tiks piešķirta lasīšanas / rakstīšanas piekļuve visām Git krātuvēm, kas atlasītas zemāk esošajā laukā. Git ir avota koda kontroles sistēma, ko parasti izmanto programmatūras izstrādātāji.

Lietotāji, kuriem ir pieeja jebkurai krātuvei, varēs izmantot arī GitWeb, lai pārlūkotu avotu un izmaiņas visos šī domēna krātuvēs.
