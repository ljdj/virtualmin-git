Git girişi etkin mi?
Bu Evet olarak ayarlandığında, kullanıcıya aşağıdaki alanda seçilen tüm Git depolarına okuma / yazma erişimi verilir. Git, genellikle yazılım geliştiricileri tarafından kullanılan bir kaynak kodu kontrol sistemidir.

Herhangi bir depoya erişimi olan kullanıcılar, kaynağı taramak ve bu etki alanındaki tüm depolarda değişiklik yapmak için GitWeb'i de kullanabilir.
