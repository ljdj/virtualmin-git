Logare autentificare Git?
Când aceasta este setată pe Da , utilizatorului i se va acorda acces de citire / scriere la toate depozitele Git selectate în câmpul de mai jos. Git este un sistem de control al codului sursă, utilizat în mod obișnuit de dezvoltatorii de software.

Utilizatorii care au acces la orice depozit vor putea folosi GitWeb pentru a naviga în sursă și pentru a modifica toate depozitele din acest domeniu.
