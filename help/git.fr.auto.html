Connexion Git activée?
Lorsque ce paramètre est défini sur Oui , l'utilisateur se verra accorder un accès en lecture / écriture à tous les référentiels Git sélectionnés dans le champ ci-dessous. Git est un système de contrôle de code source, généralement utilisé par les développeurs de logiciels.

Les utilisateurs ayant accès à tous les référentiels pourront également utiliser GitWeb pour parcourir la source et les modifications apportées à tous les référentiels de ce domaine.
