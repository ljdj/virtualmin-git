Czy logowanie jest włączone?
Gdy ta opcja jest ustawiona na Tak , użytkownik otrzyma dostęp do odczytu / zapisu do wszystkich repozytoriów Git wybranych w polu poniżej. Git to system kontroli kodu źródłowego, zwykle używany przez programistów.

Użytkownicy z dostępem do dowolnych repozytoriów będą mogli także używać GitWeb do przeglądania źródła i zmian we wszystkich repozytoriach w tej domenie.
