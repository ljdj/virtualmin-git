Уключэнне ўваходу ў Git?
Калі гэта ўстаноўлена на " Так" , карыстачу будзе прадастаўлены доступ для чытання / запісу ва ўсіх сховішчах Git, выбраных у поле ніжэй. Git - сістэма кіравання зыходным кодам, якая звычайна выкарыстоўваецца распрацоўшчыкамі праграмнага забеспячэння.

Карыстальнікі, якія маюць доступ да любых сховішчаў, таксама змогуць выкарыстоўваць GitWeb для прагляду крыніцы і змяненняў ва ўсіх сховішчах у гэтым дамене.
