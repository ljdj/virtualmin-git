Git inlog aangezet?
Wanneer dit is ingesteld op Ja zal de gebruiker lees en schrijf rechten hebben tot alle Git opslagplaatsen die geselecteerd zijn in het veld hieronder. Git is een bron code controle systeem die normaal gesproken word gebruikt door software ontwikkelaars.

Gebruikers die toegang hebben tot alle opslagplaatsen kunnen ook GitWeb gebruiken om de bronnen en wijzigingen te doorzoeken van alle opslagplaatsen in dit domein.
