Git opslagplaatsen toestaan?
Wanneer deze optie is aangezet zal de eigenaar van deze virtuele server 1 of meerdere opslagplaatsen kunnen aanmaken die geassocieerd zijn met het domein voor opslag van versie gecontroleerde bron code en andere files. De opslagplaatsen die beschikbaar zijn voor iedere gebruiker kunnen worden geconfigureerd op de email / FTP gebruikers pagina.
