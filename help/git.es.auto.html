¿Inicio de sesión de Git habilitado?
Cuando se establece en Sí , el usuario tendrá acceso de lectura / escritura a todos los repositorios Git seleccionados en el campo a continuación. Git es un sistema de control de código fuente, generalmente utilizado por los desarrolladores de software.

Los usuarios con acceso a cualquier repositorio también podrán usar GitWeb para explorar la fuente y los cambios en todos los repositorios en este dominio.
