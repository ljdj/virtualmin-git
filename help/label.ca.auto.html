Voleu permetre els dipòsits de Git?
Quan aquesta opció està habilitada, el propietari d'aquest servidor virtual serà capaç de crear un o més dipòsits Git associats al domini per emmagatzemar codi font controlat amb versions i altres fitxers. Els dipòsits disponibles per a cada usuari es poden configurar a la pàgina de correu / usuaris d’FTP.
